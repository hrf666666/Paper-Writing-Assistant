## 2. Detailed Fix Plan

### Fix A1: Fix `####` heading residual (CRITICAL)

**File**: `tools/latex_converter.py`
**Problem**: Line 343 `latex.replace('#', '\\#')` executes AFTER line 241-243 heading conversion. So `#### 3.7.1` becomes `\#\#\#\# 3.7.1` and the heading regex at line 364 cannot match `\#\#\#\#`.

**Fix**: Move `#` escaping to BEFORE heading conversion, OR handle `\#` in the heading regex:

```python
# Option A: Move # escaping before heading conversion
# In markdown_to_latex(), reorder steps:
# Step 1: Protect existing LaTeX commands (existing)
# Step 2: Convert # headings to \section FIRST (before # escaping)
# Step 3: Then escape remaining # to \#
# Step 4: Continue with other conversions

# Option B: Handle already-escaped \# in heading regex
latex = re.sub(r'^(\\#){1,6}\s+(.+)$', r'\\subsubsection{\2}', latex, flags=re.MULTILINE)
```

**Recommended**: Option A - reorder the pipeline. Move special character escaping (step 14) to execute AFTER heading conversion (step 8), but protect `#` in headings first.

### Fix A2: Fix table width overflow (CRITICAL)

**File**: `tools/latex_converter.py`, function `replace_table()`

**Current code** (line 281):
```python
col_spec = 'l' * col_count
```

**Fix**: Use `tabular*` with `\textwidth` and `p{}` column types:

```python
def replace_table(match):
    table_text = match.group(0)
    rows = table_text.strip().split('\n')
    if len(rows) < 3:
        return table_text

    header = [cell.strip() for cell in rows[0].split('|') if cell.strip()]
    data_lines = []
    for line in rows[2:]:
        row = [cell.strip() for cell in line.split('|') if cell.strip()]
        if row:
            data_lines.append(row)

    if not header:
        return table_text

    col_count = len(header)

    # Use tabular* with \textwidth for automatic width adjustment
    if col_count > 4:
        # For wide tables, use smaller font and p{} columns
        col_spec = 'p{0.12\\textwidth}' * col_count
        latex_table = '\\begin{table*}[!t]\n'  # Two-column table
        latex_table += '\\small\n'
    else:
        col_spec = 'l' * col_count
        latex_table = '\\begin{table}[!t]\n'

    latex_table += '\\caption{Comparison of performance metrics.}\n'
    latex_table += '\\label{tab:comparison}\n'
    latex_table += '\\centering\n'
    latex_table += f'\\begin{{tabular*}}{{\\textwidth}}{{@{{\\extracolsep{{\\fill}}}}{col_spec}}}\n'
    latex_table += '\\toprule\n'
    latex_table += ' & '.join(header) + ' \\\\\n'
    latex_table += '\\midrule\n'

    for row in data_lines:
        padded = row[:col_count]
        while len(padded) < col_count:
            padded.append('')
        latex_table += ' & '.join(padded) + ' \\\\\n'

    latex_table += '\\bottomrule\n'
    latex_table += '\\end{tabular*}\n'
    latex_table += '\\end{table*}' if col_count > 4 else '\\end{table}'
    return latex_table
```
